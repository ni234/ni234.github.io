Welcome to my first blog entry in TECH! The goal in this section is to get Ubuntu 16.04.2 LTS (Xenial Xerus) downloaded as a dual boot along side your windows OS. Lets get right into it.

1. The first thing your going to want to do is download the program RUFUS After you get roofus installed, using it to set up our bootable usb stick will be a peice of cake! Rufus is a small program that helps format and create bootable USB flash drives, such as USB keys/pendrives, memory sticks, etc. It can be especially useful for cases where you need to create USB installation media from bootable ISOs (Windows, Linux, UEFI, etc.)

2. Next we head over to the UBUNTU and pick up the 16.4 .iso file. For this tutorial we will be setting up the desktop version and not the server version. Also note that this install will take a few minutes as we are downloading a new operating system.

Pictured above is the s etup your going to want if you are setting up a daul boot on a windows machine looks like this. But of course we are download ubuntu 16.4 NOT 16.10. The reason we dont want the newer version has to do with technical support availability.
